Lists

Markdown supports both ordered and unordered lists:

Unordered list using asterisks: * Item 1 * Item 2 * Item 3

Unordered list using dashes: - First item - Second item - Third item

Ordered list: 1. First item 2. Second item 3. Third item

Nested lists: 1. First level - Second level * Third level 2. Back to first level
